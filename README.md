# Zirkulationspumpensteuerung

Folgende Module beinhaltet das Zirkulationspumpensteuerung Repository:

- __Zirkulationssteuerung__ ([Dokumentation](Zirkulationssteuerung))
  Steuert die Warmwasser-Zirkulationspumpe bedarfsgerecht über Bewegungsmelder, Zeitfenster und Sperrzeiten und erfasst dabei Laufzeit, Energieverbrauch sowie Kosten.
